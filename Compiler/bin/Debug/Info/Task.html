Постановка задачи


Константы – это элементы данных, значения которых известны и в процессе выполнения программы не изменяются.

Для описания констант в языке FORTRAN используется служебное слово "parameter".

Формат записи: "тип_данных, parameter :: имя_константы=значение".

Примеры:

  1. Целая десятичная константа – любое десятичное число без десятичной точки со знаком или без него: "real, parameter :: number = 1".
  2. Вещественная константа с фиксированной точкой – действительное число, представленное в виде целой и дробной частей разделенных точкой: "real, parameter :: number = 1.1"
  3. Вещественная константа с плавающей точкой – действительное число, представленное в экспоненциальной форме. Для символа экспоненты используется буква 'E': "real, parameter :: number = -1e10"

    4. В связи с разработанной автоматной грамматикой G[‹Param›] синтаксический анализатор (парсер) десятичных констант будет считать верными следующие записи констант:

    1. "real, parameter :: number = .001"
    2. "real, parameter :: number = E-10"
    3. "real, parameter :: number = 001.123"
